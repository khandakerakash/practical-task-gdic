Welcome to Angular Task of GDIC

Task Description is given below:
  1. Please create a project using Angular version 12.x.x and reactive form.
  2. Create two modules such as Authentication and Employee (Use lazy loading)
  3. In Authentication module, there will be a Login Component that has two input field, Username and Password. (Use Reactive Forms, Form Validation (required))
  4. In the employee module, there will be an employee list and add new employee functionalities.
  5. In the employee list route, using resolver service is mandatory.
  6. In add to new employee, there will be three tabs where:
    1. First tab contains employee basic information like First Name (required), Last Name, Date of Birth (required, maximum date previous day), Phone (required, value must be [0-9], minimum and maximum 11 char), Gender (required)
    2. Second tab contains employee skill information like Skill Name (required), Experience in Years (required), Skill Level (Beginner, Intermediate, Advanced) (required)
    3. Third tab contains preview of employee basic information and skill
  7. Use the state management library (NGRX) to keep the form data when move one tab to another tab.
  8. Create a repository on GitHub and share the repository.
  9. For Employee List Demo API: Click Here
N.B: Please note that number 1, 5 and 7 are required to complete the task.
